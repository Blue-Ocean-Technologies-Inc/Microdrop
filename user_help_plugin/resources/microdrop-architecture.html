DropBot Python API

Connection & Properties

import dropbot as db
proxy = db.SerialProxy()  # auto-detects connected DropBot

Read-Only Attributes

proxy.properties — device info
proxy.state — current state
proxy.number_of_channels — channel count
proxy.state_of_channels — on/off array
proxy.voltage — output voltage
proxy.frequency — output freq
proxy.hv_output_enabled — HV on?
proxy.hv_output_selected — HV/3V?

Setters

proxy.voltage = 100
proxy.frequency = 1e4
proxy.hv_output_enabled = True
proxy.hv_output_selected = True
proxy.state_of_channels = arr

state_of_channels is a 1D NumPy array (n = number_of_channels), values 0 or 1

To reconnect within a session: proxy.terminate() first to close the existing connection gracefully

Signals (Event-Driven)

Clear all: proxy.signals.clear()

DropBot Python API

Code Example — Turning on Channels

import numpy as np
import dropbot as db

# Connect to DropBot
proxy = db.SerialProxy()

# Enable HV output and set voltage/frequency
proxy.update_state(hv_output_enabled=True, hv_output_selected=True,
                  voltage=100, frequency=10e3)

# Set channel states (1 = on, 0 = off)
channels = np.zeros(proxy.number_of_channels, dtype=int)
channels[[5, 12, 23]] = 1  # activate specific electrodes
proxy.state_of_channels = channels

↓
2 Frontend publishes electrodes_state_change — toggles on/off state
↓
3 Redis / Dramatiq routes message to DropBot Controller worker
↓
4 Backend calls proxy.state_of_channels = [...] via SerialProxy
↓
5 DropBot firmware applies high voltage to selected electrodes
↑ feedback
6 Capacitance data + electrode status sent back through the message pipeline
↑
7 GUI updates with real-time sensor feedback and electrode status
09 / 10
Design

Key Design Choices

Decoupled Frontend / Backend / Server

All three can run on separate machines. Redis can be local, on LAN, or cloud-hosted. Frontend controls the backend over the network.

Plugin Architecture

Envisage plugins are independent and swappable. Features added or removed without modifying core application code.

Convention-Based Routing

Topic names map to handler methods automatically. electrodes_state_change → on_electrodes_state_change_request()

Thread Safety

Proxy transaction locks, Dramatiq worker isolation, Redis for shared state. with proxy.transaction_lock: for safe concurrent access.
